# @starknet-agentic/skill-anonymous-wallet

Starknet Anonymous Wallet skill for AI agents. Privacy-focused wallet creation and management using Typhoon SDK for shielded transactions on Starknet.

## Install

```bash
npm install @starknet-agentic/skill-anonymous-wallet
```

## What's Included

- **SKILL.md** — Full skill documentation with privacy wallet patterns
- **scripts/** — Runnable examples (create-account)

## Quick Start

```bash
node scripts/create-account.js
```

## Features

- Privacy-focused account creation
- Shielded transactions via Typhoon protocol
- Anonymous wallet management patterns
