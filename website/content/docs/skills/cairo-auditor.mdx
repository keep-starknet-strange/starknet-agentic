# Cairo Auditor

The `cairo-auditor` skill is the repo's security-review orchestrator for Cairo and Starknet code. It combines deterministic preflight checks with parallel audit passes and a strict merge/deduplication gate so the final report stays actionable.

## Installation

### Codex

```bash
npm install -g skill-installer
skill-installer install https://github.com/keep-starknet-strange/starknet-agentic/tree/main/skills/cairo-auditor
```

### Claude Code Marketplace Bundle

```text
/plugin marketplace add keep-starknet-strange/starknet-agentic
/plugin install starknet-agentic-skills@starknet-agentic-skills --scope user
/reload-plugins
/starknet-agentic-skills:cairo-auditor
```

### Agent Skills CLI

```bash
npx skills add keep-starknet-strange/starknet-agentic/skills/cairo-auditor
```

<Callout type="info" title="Claude Code scope">
Use <code>--scope user</code> as the default. Use <code>--scope local</code> only when you intentionally want a repo-specific pinned plugin state.
</Callout>

## 30-Second Happy Path

Run one deep audit and verify that the full specialist pipeline executed.

Copy the `WORKDIR=...` line from the auditor output first, then point the verification commands at that exact directory.

```text
# Codex prompt
Use cairo-auditor in deep mode on src/lib.cairo --file-output.
Output only the final report.
```

```text
# Claude Code command
/starknet-agentic-skills:cairo-auditor deep src/lib.cairo --file-output
```

```bash
# Verification
cat "$WORKDIR/cairo-audit-host-capabilities.json"
wc -l "$WORKDIR"/cairo-audit-agent-*-bundle.md
ls -lt security-review-*.md | head -n 1
```

Expected markers: `Execution Integrity: FULL`, `## Execution Trace`, Agent 1-4 vector rows, and Agent 5 adversarial row.

## Deterministic Demo Target

If you want a stable live-demo contract, clone this repository and audit:

```text
skills/cairo-auditor/tests/fixtures/insecure_upgrade_controller/src/lib.cairo
```

This fixture is intentionally vulnerable and should reliably produce three findings: missing access control on upgrade, missing timelock, and missing non-zero class-hash guard.

## Modes

| Mode | Use it when |
|------|-------------|
| `default` | Standard repo or contract review before merge |
| `deep` | High-risk changes need adversarial reasoning on top of the default pass |
| `file-targeted` | You want to re-check one or two contracts after a fix |

## What the Skill Does

1. discovers in-scope Cairo files
2. runs deterministic preflight and local benchmark-backed checks
3. spawns vector-specialist audit passes in parallel
4. deduplicates root causes and emits a single report with confidence gating

## Deterministic Local Scan

```bash
python3 skills/cairo-auditor/scripts/quality/audit_local_repo.py \
  --repo-root /path/to/your/cairo-repo \
  --scan-id my-audit
```

<Callout type="warning" title="Deterministic scans are not full audits">
Deterministic scorecards are regression gates, not proof of full audit sufficiency. Use them to catch drift, then run the full skill for release-critical work.
</Callout>
