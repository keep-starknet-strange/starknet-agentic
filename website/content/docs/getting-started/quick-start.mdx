Get your AI agent running on Starknet in **under 5 minutes**. This guide walks you through setup, your first balance query, and connecting to an AI assistant.

## Prerequisites

Before you begin, make sure you have:

- **Node.js 18+** installed ([download](https://nodejs.org/))
- **pnpm** package manager (`npm install -g pnpm`)
- A **Starknet wallet** with testnet tokens ([get tokens](https://starknet-faucet.vercel.app/))

<Callout type="tip" title="New to Starknet?">
  We recommend using [ArgentX](https://www.argent.xyz/argent-x/) browser extension. It's the easiest way to create a Starknet wallet and get testnet tokens.
</Callout>

## Step 1: Clone and Install

<Steps>
<Step number={1} title="Clone the repository">


```bash
pnpm install
```

<Step number={3} title="Build packages">

```bash
pnpm build
```

</Step>
</Steps>

## Step 2: Configure Environment

Create a `.env` file in the repository root:

```bash
cp .env.example .env
```

Edit `.env` with your Starknet credentials:

```env
# Starknet RPC endpoint (get a free key from Alchemy or Infura)
STARKNET_RPC_URL=https://starknet-sepolia.g.alchemy.com/v2/YOUR_KEY

# Your Starknet account address
STARKNET_ACCOUNT_ADDRESS=0x...

# Your account private key (NEVER share or commit this!)
STARKNET_PRIVATE_KEY=0x...
```

<Callout type="warning" title="Security Warning">
  Never commit your private key to version control. The `.env` file is already in `.gitignore`, but double-check before pushing.
</Callout>

### Getting Your Credentials

**Option 1: ArgentX Wallet (Recommended)**

1. Install [ArgentX browser extension](https://www.argent.xyz/argent-x/)
2. Create a new wallet or import existing
3. Switch to **Sepolia testnet** in settings
4. Export private key: Settings → Account → Export Private Key
5. Copy your account address from the wallet

**Option 2: Get a Free RPC Endpoint**

- [Alchemy](https://www.alchemy.com/) - Create account → New App → Starknet Sepolia
- [Infura](https://infura.io/) - Create account → New Project → Starknet
- [Blast API](https://blastapi.io/) - Free tier available

## Step 3: Run Your First Query

Check your ETH balance:

```bash
cd packages/starknet-mcp-server
pnpm run example:balance
```

You should see output like:

```

Balance: 0.5 ETH

```

## Step 4: Connect to Claude (Optional)

If you want to use Starknet tools directly in Claude Desktop:

### Configure MCP Server

Add this to your Claude Desktop config (`~/Library/Application Support/Claude/claude_desktop_config.json` on macOS):

```json
{
  "mcpServers": {
    "starknet": {
      "command": "node",
      "args": ["/path/to/starknet-agentic/packages/starknet-mcp-server/dist/index.js"],
      "env": {
        "STARKNET_RPC_URL": "https://starknet-sepolia.g.alchemy.com/v2/YOUR_KEY",
        "STARKNET_ACCOUNT_ADDRESS": "0x...",
        "STARKNET_PRIVATE_KEY": "0x..."
      }
    }
  }
}
```

### Test the Connection

Restart Claude Desktop, then try:

> "Check my STRK balance"

Claude will use the `starknet_get_balance` tool to fetch your balance.

## Troubleshooting

**"Cannot find module" error**

Make sure you ran `pnpm build` after installation.

**"Invalid account address" error**

Check that your `STARKNET_ACCOUNT_ADDRESS` is a valid Starknet address (starts with `0x`, 64 hex characters).

**"Insufficient balance" error**

Get testnet tokens from the [Starknet Faucet](https://starknet-faucet.vercel.app/).

**RPC connection timeout**

Verify your `STARKNET_RPC_URL` is correct and your API key is valid.
